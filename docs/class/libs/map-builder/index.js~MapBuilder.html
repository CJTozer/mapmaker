    MapBuilder

    MapBuilder.

    Class for encapsulating a map building operation.

    Example:

    new MapMaker()
  .spec('examples/france.yaml')
  .onError((err) => {
    console.log(err);
  })
  .onSuccess((data) => {
    console.log("Success");
    do_something_with_svg_data(data);
  })
  .build_map();
